# Ticket 041c — Réécriture README en anglais

## User story

En tant que développeur international découvrant lean-jira sur GitHub, je veux lire
la documentation d'installation et d'utilisation en anglais, afin de pouvoir configurer
et utiliser l'outil sans barrière de langue.

## Solution retenue

Réécrire `README.md` en anglais avec le même contenu que l'actuel (installation,
configuration, commandes, catalogue métriques, architecture). L'actuel `README.md`
devient `README.fr.md` — lien depuis le header du README anglais pour les utilisateurs
francophones.

## Estimation

**Bucket** : S

**Justification** : traduction mécanique d'un document existant (~300 lignes), aucun code
touché. 1 fichier renommé, 1 nouveau fichier. Aucun scénario de test. Risque bas.
